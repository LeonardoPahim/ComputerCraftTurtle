#  Computer Craft Turtle Program

Programa para uso nas _Mining Turtles_ do mod _CC: Tweaked_.

- CC: Tweaked - x.x.x
- Minecraft - 1.16.5

## Linguagens

- Lua - Mining Turtles
- TypeScript - Servidor
- React(?) - GUI
